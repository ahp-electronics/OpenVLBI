Very Long Baseline Interferometry API

OpenVLBI is a free, open source library for very long baseline interferometry.
OpenVLBI provides functions to locate, aim, cross-correlate and synchronize more nodes, or observatories together. Each observation creates a Fourier plane filled with the perpective projection of each baseline during the various captures. The nodes can contain a location companion for each sample for moving baseline projection, or a single location. Projection is calculated by an aperture synthesis algorithm. OpenVLBI saves each observation into an internal models list, where you can load your own models also. There are various model operations, including differential Fourier transform, inverse Fourier transform, shift, and masking. More operations can be done on models by using the DSP API, included into OpenVLBI. Once your work is done, the models can also be saved as jpeg, png, or fits picture files. More...
